Binding keyboard media keys for mpv.
This command is only for Linux.
Will only work if mpv socket is enabled. yt-playlist enables mpv socket.

Play/Pause command.
echo cycle pause | socat - "/tmp/mpvsocket"

Next track command.
echo playlist-next | socat - "/tmp/mpvsocket"

Prev track command.
echo playlist-prev | socat - "/tmp/mpvsocket"
