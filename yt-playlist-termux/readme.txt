Termux install.

Dependencies will be automatically installed. 

Download and install yt-playlist:

git clone https://github.com/darnrain/yt-playlist.git ${HOME}/yt-playlist/; cd ${HOME}/yt-playlist/yt-playlist-termux/; chmod +x ./install.sh; ./install.sh

to just type y to run yt-playlist

echo "alias y='yt-playlist'" >> ${HOME}/.bashrc; source ${HOME}/.bashrc

To uninstall copy yt-playlist.zip folder to your internel sorage on the phone.

Download and uninstall yt-playlist:
git clone https://github.com/darnrain/yt-playlist.git ${HOME}/yt-playlist/; cd ${HOME}/yt-playlist/yt-playlist-termux/; chmod +x ./uninstall.sh; ./uninstall.sh
